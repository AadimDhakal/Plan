# Flowchart Planner

A minimal HTML5 canvas for sketching flowcharts, logic, and project ideas.

## Features

* Blank drawing canvas
* Single black pencil tool
* Mouse support
* Touch support
* Save drawing as a PNG
* No libraries or dependencies
* Runs entirely in the browser

## Usage

1. Open `index.html` in any modern web browser.
2. Draw your flowchart or project plan using your mouse or touch screen.
3. Click **Save** to download your drawing as a PNG image.

## Purpose

This project is intended as a simple personal planning tool for:

* Flowcharts
* Project planning
* System architecture sketches
* Algorithm design
* Brainstorming ideas before development

The goal is to keep the application lightweight and distraction-free, with only the essentials needed for quick planning.

## Technologies

* HTML5
* CSS3
* JavaScript
* HTML5 Canvas API
